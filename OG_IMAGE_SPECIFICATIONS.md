# Open Graph Image Specifications
**For Design Team**
**Bright Ears Entertainment Booking Platform**
**Deadline:** Before Production Launch

---

## Overview

We need 11 Open Graph images for social media sharing (Facebook, Twitter, LinkedIn, WhatsApp, LINE). These images appear when someone shares a Bright Ears page link.

**Purpose:** Increase click-through rates from social media by 15-20%

---

## Technical Specifications

### Required Dimensions
- **Size:** 1200 x 630 pixels (Facebook/Twitter optimal)
- **Aspect Ratio:** 1.91:1
- **Safe Zone:** Keep important text/elements within 1200 x 600px (avoid top/bottom 15px)

### File Requirements
- **Format:** JPG (preferred for smaller file size) or PNG
- **File Size:** Maximum 300KB per image
- **Color Mode:** RGB
- **Resolution:** 72 DPI (web standard)

### Naming Convention
```
og-image-[page-name].jpg

Examples:
- og-image-home.jpg
- og-image-artists-listing.jpg
- og-image-corporate.jpg
```

### Save Location
```
/public/og-images/
```

## Brand Guidelines

### Primary Colors
- **Brand Cyan:** #00bbe4 (Primary CTA, accents)
- **Deep Teal:** #2f6364 (Dark backgrounds, headers)
- **Earthy Brown:** #a47764 (Secondary accents)
- **Soft Lavender:** #d59ec9 (Highlights - use sparingly)

### Secondary Colors
- **Off White:** #f7f7f7 (Backgrounds)
- **Pure White:** #ffffff (Text on dark backgrounds)
- **Dark Gray:** #333333 (Body text)

### Typography
- **Headlines:** Playfair Display (serif) - Bold/SemiBold
- **Body Text:** Inter (sans-serif) - Regular/Medium
- **Thai Text:** Noto Sans Thai (if needed)

### Logo
- Use existing Bright Ears logo
- Position: Top-left or center
- Size: 150-200px width
- Version: White logo on dark backgrounds, dark logo on light backgrounds

---

## Design Template

### Layout Structure
```
+------------------------------------------------+
|  [Logo]                                        |
|                                                |
|         [LARGE HEADLINE]                       |
|         Supporting tagline or description      |
|                                                |
|         [CTA Button or Badge]                  |
|                                                |
|  Background: Brand gradient or solid color     |
+------------------------------------------------+
```

### Text Guidelines
- **Headline:** 36-48pt, Bold, Max 5 words
- **Subheadline:** 20-28pt, Regular, Max 12 words
- **CTA Text:** 16-20pt, Medium, 2-3 words
- **Contrast:** Ensure WCAG AA compliance (4.5:1 minimum)

### Visual Elements
- Use subtle gradients (brand cyan to deep teal)
- Add floating orbs/shapes for depth (match website aesthetic)
- Include icons if relevant (music notes, location pins, etc.)
- Maintain 40-60px padding on all sides

## Design Process

### Step 1: Template Creation (1 hour)
Create a master template with:
- 1200x630px canvas
- Brand colors swatches
- Typography styles
- Logo placement
- Grid/guides for safe zones

### Step 2: Batch Production (3-4 hours)
Using the template:
- Duplicate template for each image
- Replace headline/subheadline text
- Adjust background colors/gradients
- Add page-specific elements
- Ensure consistent style

### Step 3: Quality Check (30 minutes)
- Preview at actual size (1200x630px)
- Check text readability
- Verify file sizes (<300KB)
- Test contrast ratios
- Ensure logo visibility

### Step 4: Delivery (15 minutes)
- Export all 11 images as JPG
- Name according to specification
- Compress if needed (TinyPNG, ImageOptim)
- Provide in ZIP file or folder

## Example Mockups (Reference)

### Good Examples
- **Airbnb:** Simple headline, clear imagery, strong CTA
- **Stripe:** Clean design, bold typography, minimal elements

### What to Avoid
- ❌ Too much text (max 3 elements: headline, subheadline, CTA)
- ❌ Cluttered design (keep 40% empty space)
- ❌ Poor contrast (text must be readable)
- ❌ Off-brand colors (stick to approved palette)
- ❌ Low-quality stock photos (use high-res only)

## Testing & Validation

### Before Delivery, Test With:

1. **Facebook Sharing Debugger**
   - URL: https://developers.facebook.com/tools/debug/
   - Upload image, check preview
   - Ensure 1200x630px displays correctly

2. **Twitter Card Validator**
   - URL: https://cards-dev.twitter.com/validator
   - Preview how card looks on Twitter
   - Check image cropping

3. **LinkedIn Post Inspector**
   - URL: https://www.linkedin.com/post-inspector/
   - Verify image displays properly
   - Check text readability

### Success Criteria
- ✅ All text readable on mobile preview (320px wide)
- ✅ Logo visible but not dominating
- ✅ Colors match brand guidelines
- ✅ File size under 300KB
- ✅ No pixelation or blur
